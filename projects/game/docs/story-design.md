# Wizardry Schema ストーリー設計書

## コンセプト
**D→C: ミステリーから壮大な真実へ**

表面上は「悪の魔術師ワードナを討伐する冒険」だが、
真実は「1000年間世界を守り続けてきた英雄」という反転。

---

## 表面上の設定（プレイヤーが最初に知ること）

```
1000年前、狂気の魔術師ワードナが禁忌の魔導書「スキーマ」を発動。
世界に深淵への門が開き、魔物が溢れ出した。
王国は滅び、文明は崩壊した。
今、冒険者たちはワードナを討ち、スキーマを封印するため深淵に挑む。
```

---

## 真実（徐々に明かされる）

| 段階 | 真実 | 詳細 |
|------|------|------|
| 第1 | スキーマは「開いた」のではない | 深淵を「閉じるため」の術だった |
| 第2 | ワードナは英雄だった | 1000年前、すでに門は開いていた。ワードナは命を賭けて門を「半分閉じた」 |
| 第3 | 歴史は書き換えられた | 当時の権力者が責任逃れのためワードナを悪者に |
| 最終 | 深淵の底で眠るワードナ | 彼は1000年間、門を押さえ続けている。彼を「倒す」と門が完全に開く |

---

## キーキャラクター

### セラフィナ（研究者）
- **役割**: プレイヤーの案内役、真実への導き手
- **表向き**: 古代史の研究者。スキーマの真実を求めている。
- **裏設定**: ワードナの子孫。先祖の汚名を晴らしたい。

### ギルド受付（老人）
- **役割**: 導入、伏線の配置
- **特徴**: 意味深な発言をする
